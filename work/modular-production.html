Modular production

Product designers at Vox Media write scalable front-end code by doing more with less.

Working on the Revenue Executions team, I built high-fidelity advertising experiences for brands like Vox.com, Recode, The Verge, and Eater.

Initially, I focused on applying design assets from clients — PSDs, videos, code libraries, etc. — to templates via Hymnal, Vox Media’s proprietary ad tech platform. For example, Samsung supplied Vox Media with photos and graphics created for the QLED TV. My role, then, was to match and adapt the visuals to a prexisting format, in this case, the slideshow unit pictured above.

As I grew familiar with the various templated units, my responibilities shifted to non-templated units for clients requiring custom functionality. For example, the modal unit for GMC pictured above involved three overlays triggered by toggle buttons. The bulk of my work on non-templated units like this one involved developing simple, vanilla JS plugins and designing interface animations with dependency-free CSS.
