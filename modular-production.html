Modular production

Product designers at Vox Media write modular front-end code by doing more with less.

On the Revenue Executions team, I developed high-fidelity advertising experiences for digital properties like Vox.com, Recode, The Verge, Polygon, and Racked.

Initially, I focused on applying design assets from clients—PSDs, videos, code libraries, etc.—to templates on a proprietary ad tech platform. For example, Samsung supplied Vox Media with photos and graphics created for the QLED TV. My role, then, was to match and adapt the visuals to a prexisting format, in this case, the slideshow unit pictured above.

As I grew familiar with the various templated units, my responsibilities shifted to non-templated units for clients requiring custom functionality. For example, the modal unit for GMC pictured above involved three overlays triggered by toggle buttons. The bulk of my work on non-templated units like this one involved developing scalable JavaScript plugins and designing interface animations with dependency-free CSS.
