JavaScript plugin to preload HTML elements

Here is a simple JavaScript plugin for preloading an HTML element.

          
            document.body.classList.remove("noJS");
            var preLoad = document.getElementById('preLoad');
            window.onload = function() {
            
              preLoad.classList.remove('is-loading');
            
            };

Identify preloaded element. Use removable loading class to style interface transitions with CSS.

See the top level “no JS” to style a fallback for users with JavaScript disabled.
