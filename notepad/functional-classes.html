Functional layout classes

Use the functional classes below as exceptions to the element flow defined in Global styles.

Set parent elements to relative.

          
            .relative {
            
              position: relative;
              margin: 0 auto;
            
            }

Set child elements to absolute.

          
            .absolute {
            
              position: absolute;
              top: 0;
              left: 0;
              bottom: 0;
              right: 0;
              margin: auto;
            
            }

Set flexbox parent element.

          
            .flex {
            
              display: flex;
            
            }

Absolute center flexbox child element.

          
            .flex > .center {
            
              margin: auto;
            
            }

Set height to viewport.

          
            .height100 {
            
              height: 100vh;
            
            }
