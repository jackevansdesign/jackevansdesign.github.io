Global styles for rapid design in modern browsers

Set inherited styles for type. Normalize layout, flow, and block level element spacing.

          
            * {
            
              font-family: inherit;
              font-size: inherit;
              font-weight: inherit;
              line-height: inherit;
              color: inherit;
              margin: 0;
              padding: 0;
              box-sizing: border-box;
            
            }

Default to system fonts.

          
            html {
            
              font-family: -apple-system,BlinkMacSystemFont,"Segoe UI",Roboto,"Helvetica Neue",Helvetica,sans-serif;
            
            }

Define primary type styles.

          
            body {
            
              font-size: 150%;
              font-weight: 400;
              line-height: 1.5;
            
            }

Add spacing for successive block level elements.

          
            body * + * {
            
              margin-top: 1.5em;
            
            }

Define secondary type styles.

          
            h1, h2, h3, h4, h5, h6 {
            
              font-size: 1.25em;
              line-height: 1.25;
            
            }

Reset link styles.

          
            a {
            
              text-decoration: none;
              outline: 0;
              -webkit-tap-highlight-color: transparent;
            
            }

Reset navigation list styles.

          
            nav ul {
            
              list-style-type: none;
            
            }

Override margins for navigation list items.

          
            nav ul * + * {
            
              margin-top: 0;
            
            }

Scale images.

          
            img {
            
              width: 100%;
              display: block;
            
            }
